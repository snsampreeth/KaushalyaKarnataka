# KaushalyaKarnataka

An Android application developed using Kotlin in Android Studio.

## Features
- User authentication
- Firebase integration
- Modern UI design
- Real-time data handling

## Technologies Used
- Kotlin
- Firebase
- Android Studio
- XML

## Installation
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run the app
